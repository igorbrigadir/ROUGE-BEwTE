BE SUMMARY EVALUATION PACKAGE DOWNLOAD PAGE

 

 

 

The BE Package is software that automatically evaluates text summaries against ideal summaries produces by people.  It was designed by Eduard Hovy, Chin-Yew Lin, Stephen Tratz, and Liang Zhou.  This implementation was written by Stephen Tratz at USC/ISI.

The BE Package has been used in many research projects and source code can be downloaded here for non-commercial use. (Please read the license and scroll to bottom of this page).

Please address inquiries about this software to Stephen Tratz (tratz@usc.edu) or Eduard Hovy (hovy@isi.edu).
